# Jekyll In Container

Getting Started
---------------
1. install docker
2. install docker-compose
3. clone repository: `git clone --recursive https://github.com/zhao-lin-li/jekyll_in_container.git`
4. run bootstrap.sh: `./bootstrap.sh`
5. start service: `docker-compose up`
